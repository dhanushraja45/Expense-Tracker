Expense Tracker 💰

Expense Tracker is a fully responsive web application built using Django and Tailwind CSS, designed to help users manage their personal finances efficiently. The application allows users to track expenses and incomes, view detailed summaries, and export financial data in multiple formats.

This project reflects my hands-on experience in Python Full Stack Development, focusing on secure authentication, clean backend architecture, and a modern responsive UI.

✨ Features

Expenses & Incomes Dashboard
View and manage detailed expense and income records in one place.

User Authentication
Secure signup, login, logout, and user-specific data access.

CRUD Operations
Create, read, update, and delete:

Expenses

Incomes

Categories / Sources

Category-Wise Tracking
Track expenses and incomes based on categories and sources.

Data Export
Export financial data in:

📄 PDF

📊 CSV

📈 Excel

Analytics & Statistics
Monthly summaries and visual statistics for better financial insights.

Responsive UI
Fully responsive design for mobile, tablet, and desktop using Tailwind CSS.
